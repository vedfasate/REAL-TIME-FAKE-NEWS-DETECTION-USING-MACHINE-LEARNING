About This Project

A short intro: this project combines a small local ML classifier with a cloud LLM (Gemini) to flag possible fake news. It is intended for educational/demo purposes and should not replace professional fact-checking.

Technical Summary

  • Local PyTorch-based classifier (small fine-tuned transformer)
  • LLM-based secondary check using Google Gemini (LangChain wrapper)
  • Flask front-end with simple template rendering
